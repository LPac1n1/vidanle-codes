Deseja abrir um editor de código?

O Front Editor é uma ferramenta de edição de código web popular e versátil, amplamente utilizada por desenvolvedores e designers para criar e personalizar páginas web. Este editor oferece uma interface intuitiva que permite aos usuários visualizar as mudanças em tempo real à medida que fazem edições no código HTML, CSS e JavaScript.

Declaração do Tipo de Documento e Tag Raiz
1º Passo

A primeira linha é uma declaração que informa ao navegador que o documento é um arquivo HTML5. A segunda linha é a tag raiz do documento HTML e define o idioma padrão do documento como português ("pt-br").

HTML


    <!DOCTYPE html>
    <html lang="pt-br">
Cabeçalho da Página
2º Passo

O cabeçalho da página contém informações sobre o documento HTML, como o título da página, links para arquivos CSS e JavaScript, e outras informações importantes. As tags meta definem o conjunto de caracteres usado no documento, a largura do dispositivo e a escala inicial para dispositivos móveis, e informa ao Internet Explorer para usar a versão mais recente do mecanismo de renderização. As tags link vinculam os arquivos CSS e fontes externas à página HTML. A tag script vincula o arquivo JavaScript externo à página HTML e usa o atributo defer para garantir que o script seja executado apenas após o carregamento completo da página.

HTML


    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <meta http-equiv="X-UA-Compatible" content="IE=edge" >
      <title>Cronômetro</title>
      <link rel="stylesheet" href=css/styles.css>
      <script src="js/scripts.js" defer></script>
      <link rel="preconnect" href="https://fonts.googleapis.com">
      <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
      <link href="https://fonts.googleapis.com/css2?family=Orbitron:wght@400;500;600;700;800;900&display=swap" rel="stylesheet">
    </head>
Corpo da Página
3º Passo

O corpo da página contém o conteúdo visível da página. Ele começa com uma tag body que envolve todo o conteúdo da página. A tag div com a classe "container" é usada para agrupar todo o conteúdo em um único contêiner. A tag h1 define o título principal da página como "Cronômetro". A tag div com a classe "timer" é usada para exibir o cronômetro na página, enquanto as tags div com a classe "time" e "separator" são usadas para exibir os minutos, segundos e milissegundos do cronômetro. A tag div com a classe "buttons" é usada para agrupar os botões de controle do cronômetro na página. Cada botão é uma tag button com uma classe e um ID únicos.

HTML


    <body>
    <div class="container">
      <h1>Cronômetro</h1>
    
      <div class="timer">
        <div class="time" id="minutes">00</div>
        <div class="separator">:</div>
        <div class="time" id="seconds">00</div>
        <div class="separator">:</div>
        <div class="time" id="milliseconds">000</div>
      </div>
  
      <div class="buttons">
        <button class="btn" id="startBtn">Iniciar</button>
        <button class="btn" id="pauseBtn">Pausar</button>
        <button class="btn" id="resumeBtn">Continuar</button>
        <button class="btn" id="resetBtn">Resetar</button>
      </div>
    </div>
  </body>
Estilos Globais
4º Passo

Este trecho define estilos globais para todos os elementos da página. Ele remove as margens e preenchimentos padrão e define a fonte para "orbitron" ou uma fonte sans-serif.

CSS


    *{
      margin: 0;
      padding: 0;
      font-family: "orbitron", sans-serif;
    }
Estilos do Contâiner
5º Passo

Este estilo aplica um layout flexível ao elemento com a classe "container". Ele define o contêiner como uma coluna, centraliza verticalmente e horizontalmente seu conteúdo e define um espaçamento vertical entre os elementos de 2rem. A altura do contêiner é definida como 100vh, ou seja, ocupa toda a altura da janela.

CSS


    .container{
      display: flex;
      flex-direction: column;
      align-items: center;
      justify-content: center;
      gap: 2rem;
      
      height: 100vh;
    }
Estilos do Título
6º Passo

Este estilo define a cor do elemento h1 dentro do contêiner como azul (#175cff).

CSS


    .container h1{
      color: #175cff;
    }
Estilos do Cronômetro
7º Passo

Este estilo aplica um layout flexível ao elemento com a classe "timer". Ele define o cronômetro como um elemento flexível e define o tamanho da fonte como 55px, o peso da fonte como negrito e a cor do texto como cinza escuro (#333).

CSS


    .timer{
      display: flex;
      font-size: 55px;
      font-weight: bold;
      color: #333;
    }
Estilos dos Tempos
8º Passo

Esses estilos são aplicados aos elementos que exibem os tempos do cronômetro. A classe "time" define o estilo geral dos tempos, alinhando-os vertical e horizontalmente, definindo uma margem à direita, a cor do texto como azul escuro (#172953) e uma largura mínima de 90px. A classe "separator" define o estilo do separador entre os tempos, adicionando um preenchimento à esquerda e à direita e definindo a cor do texto como cinza escuro (#525763). O ID "milliseconds" define o estilo específico para os milissegundos, definindo o tamanho da fonte como 30px e a cor do texto como cinza escuro (#525763).

CSS


    .time{
      display: flex;
      align-items: center;
      justify-content: center;
      margin-right: 10px;
      color: #172953;
      min-width: 90px;
    }
    
    .separator{
      padding: 0 10px;
      color: #525763;
    }

    #milliseconds{
      font-size: 30px;
      color: #525763;
    }
Estilos dos Botões
9º Passo

Esses estilos são aplicados aos botões na página. A classe "buttons" define um layout flexível para os botões, com um espaçamento horizontal de 1rem entre eles. A classe "btn" define o estilo geral dos botões, incluindo preenchimento, tamanho da fonte, peso da fonte, cursor, borda arredondada, borda e uma transição suave de propriedades. Quando o cursor está sobre um botão, ele é ampliado em escala (scale). O ID "startBtn" define o estilo específico para o botão de iniciar, com uma cor de fundo azul (#1755ff) e cor do texto branca (#fff). Os IDs "pauseBtn" e "resumeBtn" têm o estilo display: none, tornando-os inicialmente invisíveis.

CSS


    .buttons{
      display: flex;
      gap: 1rem;
    }

    .btn{
      padding: 10px 20px;
      font-size: 18px;
      font-weight: bold;
      cursor: pointer;
      border-radius: 30px;
      border: 1px solid #ccc;
      transition: all 0.2s ease-in-out;
    }
    .btn:hover{
      transform: scale(1.1);
    }
    
    #startBtn{
      background-color: #1755ff;
      color: #fff;
    }
    #pauseBtn, #resumeBtn{
      display: none;
    }
Estilos do Corpo da Página
10º Passo

Este estilo define a cor de fundo do corpo da página como um tom de cinza escuro (rgba(19,21,21)) e define a altura do corpo como 100%.

CSS


    body{
      background: rgba(19,21,21);
      background-repeat: no-repeat;
      height: 100%;
    }
Seletores de Elementos
11º Passo

Esses seletores são usados para acessar os elementos HTML que são manipulados pelo JavaScript. Cada elemento é selecionado por seu ID usando o método document.querySelector().

JavaScript


    const minutesEl = document.querySelector("#minutes");
    const secondsEl = document.querySelector("#seconds");
    const millisecondsEl = document.querySelector("#milliseconds");
    const startBtn = document.querySelector("#startBtn");
    const pauseBtn = document.querySelector("#pauseBtn");
    const resumeBtn = document.querySelector("#resumeBtn");
    const resetBtn = document.querySelector("#resetBtn");
Definir Variáveis
12º Passo

Essas variáveis são usadas para armazenar o estado atual do cronômetro. A variável interval é usada para armazenar o ID do intervalo criado pelo método setInterval. As variáveis minutes, seconds e milliseconds são usadas para armazenar o tempo atual do cronômetro. A variável isPaused é usada para verificar se o cronômetro está pausado.

JavaScript


    let interval;
    let minutes = 0;
    let seconds = 0;
    let milliseconds = 0;
    let isPaused = false;
Adicionar Event Listeners
13º Passo

Esses ouvintes de eventos são usados para detectar quando um botão é clicado e chamar a função correspondente.

JavaScript


    startBtn.addEventListener("click", startTimer)
    pauseBtn.addEventListener("click", pauseTimer) 
    resumeBtn.addEventListener("click", resumeTimer)
    resetBtn.addEventListener("click", resetTimer)
Funções do Cronômetro
14º Passo

Essas funções são usadas para controlar o cronômetro. A função startTimer() inicia o cronômetro e incrementa o tempo a cada intervalo de 10 milissegundos, atualizando a exibição dos minutos, segundos e milissegundos no HTML. A função pauseTimer() pausa o cronômetro, definindo a variável isPaused como verdadeira. A função resumeTimer() retoma o cronômetro, definindo a variável isPaused como falsa. A função formatTime() é usada para formatar o tempo em minutos e segundos com um zero à esquerda, se necessário. A função formatMilliseconds() é usada para formatar os milissegundos com zeros à esquerda, se necessário. A função resetTimer() reinicia o cronômetro e redefine a exibição dos minutos, segundos e milissegundos no HTML.

JavaScript


    function startTimer() {
      interval = setInterval(() => {
        if(!isPaused){
          milliseconds += 10
    
          if(milliseconds === 1000){
            seconds++
            milliseconds = 0
          }
    
          if(seconds == 60){
            minutes++
            seconds = 0
          }
    
          minutesEl.textContent = formatTime (minutes);
          secondsEl.textContent = formatTime (seconds);
          millisecondsEl.textContent = formatMilliseconds(milliseconds);
        }
      
      }, 10)
    
      startBtn.style.display = "none"
      pauseBtn.style.display = "block"
    }
    
    function pauseTimer() {
      isPaused = true
      pauseBtn.style.display = "none"
      resumeBtn.style.display = "block"
    }
    
    function resumeTimer() {
      isPaused = false
      pauseBtn.style.display = "block"
      resumeBtn.style.display = "none"
    }
    
    function formatTime(time) {
      return time < 10 ? `0${time}` : time;
    }
    
    function formatMilliseconds(time){
      return time < 100 ? `${time}`.padStart(3, "0") : time;
    }
    
    function resetTimer() {
      clearInterval(interval)
      minutes = 0
      seconds = 0
      milliseconds = 0
    
      minutesEl.textContent = "00"
      secondsEl.textContent = "00"
      millisecondsEl.textContent = "000"
    
      startBtn.style.display = "block"
      pauseBtn.style.display = "none"
      resumeBtn.style.display = "none"
    }
Event Listener Adicional
15º Passo

Este ouvinte de eventos é usado para garantir que a variável isPaused seja definida como falsa quando o botão de redefinição for clicado, para evitar que o cronômetro seja iniciado em um estado pausado.

JavaScript


    resetBtn.addEventListener("click", () => {
      isPaused = false
    })
