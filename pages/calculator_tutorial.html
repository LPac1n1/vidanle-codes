Calculadora |

Deseja abrir um editor de código?

O Front Editor é uma ferramenta de edição de código web popular e versátil, amplamente utilizada por desenvolvedores e designers para criar e personalizar páginas web. Este editor oferece uma interface intuitiva que permite aos usuários visualizar as mudanças em tempo real à medida que fazem edições no código HTML, CSS e JavaScript.

HTML Básico
1º Passo

Começamos com o esqueleto básico do HTML, definindo o tipo de documento, a linguagem da página e configurando metadados no head. O título é definido como "Calculadora", e um link para o arquivo CSS externo é incluído para estilização.

HTML


    <!DOCTYPE html>
    <html lang="pt-br">
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Calculadora</title>
        <link rel="stylesheet" href="./style.css">
      </head>
      <body>
Estrutura da Página HTML
2º Passo

Dentro do body, criamos contêineres para a aparência visual. Um contêiner "fundo" com um gradiente de cores e um contêiner "calculadora" que abriga o título, um parágrafo para exibir resultados e uma tabela para os botões da calculadora.

HTML


    <div class="fundo">
    <div class="calculadora">
      <h1>Calculadora</h1>
      <p id="resultado"></p>
      <table>
Botões da Calculadora em HTML
3º Passo

Os botões da calculadora são organizados em linhas e colunas dentro de uma tabela HTML. Cada botão é um elemento button com a classe "botao" e uma função JavaScript associada para ser executada ao ser clicado.

HTML


    <tr>
      <td><button class="botao" onclick="clean()">C</button></td>
      <td><button class="botao" onclick="back()"><</button></td>

      <td><button class="botao" onclick="insert('*')">X</button></td>
    </tr>
    <tr>

      <td><button class="botao" onclick="insert('-')">-</button></td>
    </tr>
    <tr>
      <td><button class="botao" onclick="insert('4')">4</button></td>
      <td><button class="botao" onclick="insert('5')">5</button></td>
      <td><button class="botao" onclick="insert('6')">6</button></td>
      <td><button class="botao" onclick="insert('+')">+</button></td>
    </tr>
    <tr>
      <td><button class="botao" onclick="insert('1')">1</button></td>
      <td><button class="botao" onclick="insert('2')">2</button></td>

Terminar a estrutura HTML
4º Passo

Fechamento da tabela de botões e a inclusão do script JavaScript no final do body para manipular a lógica da calculadora.

HTML


          </table>

        </div>
    
        <script src="./script.js"></script>

Estilização com CSS
5º Passo

O CSS começa com um reset de margens e preenchimentos para garantir consistência entre os navegadores.

CSS


    * {
      margin: 0;
      padding: 0;
    }
Estilo do Fundo e da Calculadora
6º Passo

A classe "fundo" estiliza o gradiente de fundo, definindo altura, cor do texto e fonte.

CSS


    .fundo {
      background-image: linear-gradient(
        45deg,
        rgba(19, 21, 21, 1),
        rgba(0, 51, 206, 1),
        rgba(19, 21, 21, 1)
      );
      height: 100vh;
      color: white;
      font-family: Arial, Helvetica, sans-serif;
      text-align: center;
    }
Estilo da Calculadora
7º Passo

A classe "calculadora" posiciona a calculadora no centro da tela, define sua aparência e adiciona bordas arredondadas.

CSS


    .calculadora {
      position: absolute;
      background-color: rgba(0, 0, 0, 1);
      top: 50%;
      left: 50%;
      transform: translate(-50%, -50%);
      border-radius: 12px;
      padding: 12px;
    }
Estilo dos Botões
8º Passo

A classe "botao" estiliza os botões da calculadora com tamanho, fonte e cores específicas.

CSS


    .botao {
      width: 50px;
      height: 50px;
      font-size: 25px;
      margin: 3px;
      cursor: pointer;
      background-color: rgb(31, 31, 31);
      border: none;
      color: white;
    }
Efeito de Hover nos Botões
9º Passo

Ao passar o mouse sobre os botões, o fundo escurece para fornecer feedback visual.

CSS


    .botao:hover {
      background-color: black;
    }
Estilo do Visor de Resultados
10º Passo

A estilização do visor de resultados inclui cor de fundo, tamanho, margens e alinhamento do texto.

CSS


    #resultado {
      background-color: white;
      width: 207px;
      height: 30px;
      margin: 5px;

    }
Lógica da Calculadora com JavaScript
11º Passo

No JavaScript, a função insert adiciona números e operadores ao visor.

JavaScript


    function insert(num) {
      var numero = document.getElementById("resultado").innerHTML;
      document.getElementById("resultado").innerHTML = numero + num;
    }
Limpar Visor com JavaScript
12º Passo

No JavaScript, a função insert adiciona números e operadores ao visor.

JavaScript


    function clean() {
      document.getElementById("resultado").innerHTML = "";
    }
Retornar um Caractere com JavaScript
13º Passo

A função back remove o último caractere do visor.

JavaScript


    function back() {
      var resultado = document.getElementById("resultado").innerHTML;
      document.getElementById("resultado").innerHTML = resultado.substring(
        0,
        resultado.length - 1
      );
    }
Calcular com JavaScript
14º Passo

A função calcular avalia a expressão no visor e exibe o resultado ou uma mensagem de erro. Essas funções são vinculadas aos botões HTML para interatividade.

JavaScript


    function calcular() {
      var resultado = document.getElementById("resultado").innerHTML;
      if (resultado) {
        document.getElementById("resultado").innerHTML = eval(resultado);
      } else {
        document.getElementById("resultado").innerHTML = "Nada a calcular...";
      }
    }
