Deseja abrir um editor de código?

O Front Editor é uma ferramenta de edição de código web popular e versátil, amplamente utilizada por desenvolvedores e designers para criar e personalizar páginas web. Este editor oferece uma interface intuitiva que permite aos usuários visualizar as mudanças em tempo real à medida que fazem edições no código HTML, CSS e JavaScript.

Estrutura Básica
1º Passo

O código começa com as tags !DOCTYPE html e html, que definem o tipo de documento e iniciam o elemento HTML, respectivamente. Em seguida, temos a tag head, onde são definidas informações sobre a página, como o título exibido na aba do navegador e as referências a arquivos externos, como o arquivo CSS e o arquivo JavaScript. O conteúdo visível da página é colocado dentro da tag body.

HTML


    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8" />
      <meta http-equiv="X-UA-Compatible" content="IE=edge" />
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
      <link rel="stylesheet" href="./styles.css" />
      <title>Jogo da Velha</title>
    </head>
    <body>
    <!-- Conteúdo visível da página -->
    </body>
    </html>
Formulário de Nomes dos Jogadores
2º Passo

Dentro da div class="players-name", há um formulário para inserir os nomes dos jogadores. Existem dois campos de texto, cada um com um rótulo (label) e um espaço reservado (placeholder). Além disso, há um botão "Começar" (input type="submit") para iniciar o jogo

        <input type="text" placeholder='Será o "O"' name="second-player" id="second-player">

      <div>

Criar Placar
3º Passo

A div class="score-box" contém o placar do jogo. Ela exibe os nomes dos jogadores e suas pontuações atuais. Os nomes dos jogadores são exibidos nas tags p class="player-name", enquanto as pontuações são exibidas nas tags span class="score-number".

      <div class="score-box">
      <div class="flex">
        <p class="player-name" id="first-player-name"></p>
        <span class="score-number" id="first-player-score">0</span>
      </div>

      <div class="flex">
        <p class="player-name" id="second-player-name"></p>
        <span class="score-number" id="second-player-score">0</span>
      </div>
    </div>
Tabuleiro do Jogo
4º Passo

A div class="board" representa o tabuleiro do jogo. Cada célula do tabuleiro é representada por uma div class="cell". As células são identificadas pelo atributo data-cell.

Mensagem de Vitória
5º Passo

A div class="winning-message" é exibida quando um jogador vence o jogo. Ela contém uma mensagem de vitória (p class="winning-message-text") e um botão "Reiniciar" (button class="winning-message-button") para reiniciar o jogo.

HTML


    <div class="winning-message" data-winning-message>
      <p class="winning-message-text" data-winning-message-text>X Venceu!</p>
      <button class="winning-message-button" data-restart-button>
        Reiniciar!
      </button>
    </div>
Adicionar o Script
6º Passo

No final do código, há uma tag script que faz referência a um arquivo JavaScript externo (scripts.js). Esse arquivo contém a lógica do jogo, como a detecção de vitória e a manipulação dos elementos da página.

HTML


    <script src="./scripts.js"></script>

Adicionar Evento de Clique
22º Passo

O código define um evento de clique para o botão de início do jogo usando startButton.onclick. Quando o botão é clicado, algumas classes são adicionadas e removidas para mostrar e ocultar os elementos corretos na página. Além disso, a função showNamesOnScore() é chamada para exibir os nomes dos jogadores no placar.

JavaScript


    startButton.onclick = () => {
      playersName.classList.add("hide")
      game.classList.remove("hide")
  
      showNamesOnScore()
    }

Funções do Jogo
24º Passo

O código define várias funções para controlar o jogo. A função startGame() é chamada para iniciar o jogo, onde as células do tabuleiro são configuradas e os eventos de clique são adicionados. A função endGame() é chamada quando o jogo termina, exibindo uma mensagem de vitória ou empate. Outras funções, como checkForWin(), checkForDraw(), placeMark(), setBoardHoverClass(), swapTurns() e handleClick(), são responsáveis por verificar a vitória, empate, marcar as células, alternar entre os jogadores e lidar com os cliques nas células.

JavaScript


    const startGame = () => {
    isCircleTurn = false;

    for (const cell of cellElements) {
      cell.classList.remove("circle");
      cell.classList.remove("x");
      cell.removeEventListener("click", handleClick);
      cell.addEventListener("click", handleClick, { once: true });
    }

    setBoardHoverClass();
    winningMessage.classList.remove("show-winning-message");
  };
  
  const endGame = (isDraw) => {
    if (isDraw) {
      winningMessageTextElement.innerText = "Empate!";
    } else {
      winningMessageTextElement.innerText = `${isCircleTurn ? "O" : "X"} venceu!`;
      isCircleTurn ? oScore++ : xScore++
      showScore()
    }

    winningMessage.classList.add("show-winning-message");
  };
Adicionar Pontuação
25º Passo

O código mantém a pontuação dos jogadores usando as variáveis xScore e oScore. As pontuações são atualizadas na função endGame() e exibidas no placar usando as variáveis firstPlayerScoreText e secondPlayerScoreText.

JavaScript


    let xScore = 0
    let oScore = 0
    
    const showNamesOnScore = () => {
      if (firstPlayerName.value == "") {
        firstPlayerName.value = "Jogador 1"
      }
      if (secondPlayerName.value == "") {
        secondPlayerName.value = "Jogador 2"
      }
  
      firstScoreName.innerHTML = firstPlayerName.value
      secondScoreName.innerHTML = secondPlayerName.value
    }
    
    const showScore = () => {
      firstPlayerScoreText.innerText = xScore
      secondPlayerScoreText.innerText = oScore
    }
